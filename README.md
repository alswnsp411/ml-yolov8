# YOLO v8 (You Only Look Once) Object Detection and Tracking
<br/>

### 프로젝트 개요 
YOLOv8을 사용하여 세 가지 카테고리 ('봉지라면', '음료수', '컵라면')의 객체를 검출하고 추적한다. 
Roboflow에서 데이터를 수집하고, Google Colab을 활용하여 모델을 학습시키며, 학습된 모델을 사용하여 객체 검출 및 추적 결과를 비디오로 생성한다.
